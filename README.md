# Linear-Todo-Manager

- 一个单 Python 脚本的线性 todo 管理工具 | A single Python script todo management tool
- [EN](https://github.com/haeob/Linear-Todo-Manager/blob/main/README_EN.md)
## 特点
- 线性的todo列表。（并始终展示当前todo） 
- 有专注模式，和番茄钟🍅（倒计时和进度条）
- 有可自定义激励语句。
---
- 使用简单。
- 无需安装，直接运行单个脚本。
- 自动创建文件并存储本地数据。
- 免费
  
## 使用
> 需要安装 python。
- 直接在CMD中运行脚本即可。
